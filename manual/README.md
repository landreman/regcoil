# Not the canonical docs

This LaTeX manual describes the legacy Fortran executable and namelist input
format. As of Phase 12 of the Python migration
([`docs/migration/PHASES.md`](../docs/migration/PHASES.md)), it is **no
longer the canonical user documentation**.

For current documentation (installation, tutorials, every plot type, and the
API reference for the Python package), see
**[regcoil.readthedocs.io](https://regcoil.readthedocs.io)** or build it
locally from [`docs/`](../docs/).

This directory is kept for historical reference (and for the REGCOIL paper
figures) until Phase 13 removes the legacy Fortran executable entirely.
